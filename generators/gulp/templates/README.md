# templates-gulp
> Templates of web generator.

## Generator

* [generator-web](https://github.com/lixinliang/generator-web)

## Role

* Gulp
    * Watch files.
* Rollup
    * Rollup handles `.js` files.
* Postcss
    * Postcss handles `.css` files.

## Usage

* `$ yo web:gulp`

## Cli

* dev: `$ npm run dev`
* build: `$ npm run build`
* dist: `$ npm run dist`
