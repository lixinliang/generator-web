# generator-web
> 🎉A simple web generator.

## 简单、粗暴的web脚手架，易扩展。

### Env

* Node - v4.7.0
* npm - 2.15.11

### Installation

* Clone or download.
* `$ cd generator-web`
* `$ npm install`
    * If install fail, please try again later.
